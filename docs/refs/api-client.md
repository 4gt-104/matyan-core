# Client API (matyan-client)

Public Python API of the **matyan-client** package. Install with `pip install matyan-client` (or from the repo) and import from `matyan_client`.

## Main classes

::: matyan_client.Run
    options:
      show_source: false
      show_root_heading: true

::: matyan_client.Repo
